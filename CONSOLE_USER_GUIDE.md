# EvoMap-Lite Console 用户指南

## 概述

EvoMap-Lite Console 是 Web 管理界面，用于：
- 浏览和搜索资产
- 审核候选资产
- 管理节点
- 查看系统统计和趋势

## 功能模块

### 1. 资产浏览（Asset Browser）

访问路径：`/assets`

**功能**：
- 按项目、命名空间筛选
- 搜索资产
- 切换是否显示候选资产
- 查看资产详情
- 按置信度、成功率排序

**使用场景**：
- 查找可复用的 Gene/Capsule
- 检查资产质量指标
- 浏览特定项目的资产

### 2. 审核台（Review Board）

访问路径：`/review`

**功能**：
- 查看待审核的候选资产
- 审批资产（Promote）
- 拒绝资产（Reject）
- 填写拒绝理由

**使用场景**：
- 质量审核
- 风险评估
- 决策记录

### 3. 节点面板（Node Panel）

访问路径：`/nodes`

**功能**：
- 查看所有注册节点
- 查看节点状态（Active/Inactive）
- 查看节点能力（Capabilities）
- 查看节点统计（Genes/Capsules 数量）

**使用场景**：
- 监控节点健康度
- 管理节点配额
- 审计节点活动

### 4. 仪表盘（Dashboard）

访问路径：`/`

**功能**：
- 系统概览统计
- 资产总数、成功/失败数
- 系统运行时间
- 流量和成功率趋势图

**使用场景**：
- 系统健康监控
- 性能趋势分析
- 使用量统计

## 工作流程

### 典型审核流程

1. **访问审核台**：导航到 `/review`
2. **查看候选资产**：浏览所有待审核的资产
3. **查看资产详情**：点击资产查看完整信息
4. **决策**：
   - 如果符合要求，点击 **Approve**
   - 如果不符合，点击 **Reject** 并填写理由
5. **确认决策**：系统会弹出确认对话框
6. **查看结果**：资产状态会更新（Promoted/Rejected）

### 典型资产搜索流程

1. **访问资产浏览**：导航到 `/assets`
2. **设置筛选条件**：
   - 输入项目名称
   - 输入命名空间
   - 切换候选资产显示
3. **搜索**：点击搜索按钮
4. **查看结果**：浏览匹配的资产
5. **排序**：点击表头按置信度或成功率排序
6. **查看详情**：点击资产查看完整信息

## 常见问题

### Q: 为什么看不到候选资产？

A: 检查右上角的 **Include Candidate** 开关是否已打开。

### Q: 审批后资产状态没有更新？

A: 刷新页面或等待几秒钟，系统需要时间处理决策。

### Q: 节点显示为 Inactive？

A: 节点可能已断开连接或超过超时时间。检查节点日志。

### Q: 趋势图没有数据？

A: 确保后端 `/stats` API 正常返回数据。检查网络请求。

## 最佳实践

1. **定期审核**：每天检查候选资产，及时处理
2. **使用筛选**：使用项目/命名空间筛选快速定位相关资产
3. **查看详情**：决策前查看资产详情，了解完整信息
4. **填写拒绝理由**：拒绝时填写详细理由，便于后续分析
5. **监控节点**：定期检查节点面板，确保系统健康
